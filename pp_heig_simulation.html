pp_heig_simulation.apply_power_profile(net: pandapowerNet, equipment: str, power_profiles: dict[str, DataFrame])[source]#

Apply power profiles stored in dictionary of a pandas DataFrame format to a pandapower network in order to perform a pandapower time simulation powerflow docs.

The function creates pandapower Controller for each DataFrame found in power_profiles input. Note that if the network already contains controllers linked to the equipment input, the function will first erase them. In order to apply wanted power profiles to wanted element, use profile_mapping column found in load and sgen equipments.

Parameters:

  • net (pandapower.pandapowerNet) – pandapower network object

  • equipment (str) – The equipment name in which power profiles will be applied (load, sgen).

  • power_profiles (dict[str, pd.DataFrame]) – These are power profiles stored in a dictionary in DataFrame pandas format. The keys of the dictionary correspond to the power parameter of equipments (p_mw, q_mvar).

Example

>>> apply_power_profile(net=net, equipment="load", power_profiles=timeseries["load"])
pp_heig_simulation.create_output_writer(net: pandapowerNet, add_results: [list[str] | str] = None)[source]#

Define which results will be stored as output.

The function create a pandapower OutputWriter with the following default outputs:

  • res_bus.vm_pu

  • res_line.loading_percent

  • res_trafo.loading_percent

Other needed outputs could be added using add_results parameters.

Parameters:

  • net (pandapower.pandapowerNet) – Specify a pandapower network object.

  • add_results ([list[str] | str], None) – Store the additional simulation results as output.

Example

>>> create_output_writer(net=pp.pandapowerNet)
pp_heig_simulation.load_net_from_xlsx(file_path: str) → pandapowerNet[source]#

Create a pandaPower network object using data stored in a xlsx file.

Pay attention to the following points:

  • Xlsx file template should be the one provided in the package repository.

  • Every column marked in green is mandatory.

  • Every column marked in yellow is mandatory for single phase short-circuit simulation.

  • Every column marked in red is mandatory to map loads and generators to their timeseries power profile timeseries powerflow simulation.

Parameters:

file_path (str) – File path of the xlsx file where the power network data are stored.

Returns:

net – Create pandaPower object from xlsx file.

Return type:

pandapower.pandapowerNet

Example

>>> pp.pandapowerNet = load_net_from_xlsx(file_path="net_file_path.xlsx")
pp_heig_simulation.load_power_profile_form_xlsx(file_path: str) → dict[str, dict[str, DataFrame]][source]#

Load power profile from a xlsx file and return a dictionary of pandas DataFrame ready to be applied to a pandaPower network for timeseries simulations.

The function also interpolate power profiles in order they all have the same time steps. Note that the function is only able to load one-day power profiles.

Parameters:

file_path (str) – File path of the xlsx file where the power network data are stored.

Returns:

power_profile – Power profiles stored in dictionary of pandas DataFrame format.

Return type:

dict[str, dict[str, pd.DataFrame]]

Example

>>> load_power_profile_form_xlsx(file_path="profile_file_path")
pp_heig_simulation.run_time_simulation(net: pandapowerNet, output_filename: str | None = None, folder: str = 'output') → dict[str, DataFrame][source]#

Run a time-series power flow on a pandaPower network where power profiles has been applied to the load and the generators.

The results will then be converted into a dictionary of DataFrames and saved to an Excel file if desired.

Parameters:

  • net (pandapower.pandapowerNet) – Specify a pandapower network object containing two elements: Controller; OutputWriter.

  • output_filename (str, None) – Write the name of the .xlsx file where the simulation results will be stored. If this parameter is not filled, the function will not save the results.

  • folder (str, "output") – It is the folder name where results will be stored.

Example

>>> run_time_simulation(net=net, folder=".cache", output_filename="output_filename")
